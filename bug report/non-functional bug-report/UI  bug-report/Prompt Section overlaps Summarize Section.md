__Title:__ Prompt Section overlaps Summarize Section

__Description__

When the user opens the Prompt section and then scrolls/clicks to Summarize section, the Summarize section is visually hidden behind the Prompt section. Users cannot see or interact with the Summarize options.

__Precondition:__

1.   User is logged in on chat dashboard

__Steps:__

1. Open https://getbot.ai/dashboard/chat 
2. on iPhoneTap Prompt → Prompt section expands
3. Tap or scroll to Summarize section
4. Observe that Summarize section is overlapped by Prompt section


__Actual result:__

Prompt section covers Summarize section
Users cannot interact with Summarize section

__Expected result:__

• Summarize section should appear fully visible above Prompt section
• Users can see and interact with all options
