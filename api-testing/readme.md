# ­Ъџђ API Testing Project: JSON-Server & Postman Automation

__рЃћрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА API рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃАрЃарЃБрЃџ рЃфрЃўрЃЎрЃџрЃА: рЃџрЃЮрЃЎрЃљрЃџрЃБрЃарЃў рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃњрЃљрЃЏрЃљрЃарЃЌрЃЋрЃўрЃЊрЃљрЃю рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃбрЃћрЃАрЃбрЃћрЃЉрЃўрЃА рЃАрЃЎрЃарЃўрЃърЃбрЃўрЃарЃћрЃЉрЃљрЃЏрЃЊрЃћ.__

## ­ЪЏа рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў
* **Backend Simulation:** [JSON-Server](https://github.com/typicode/json-server) (Node.js)
* **API Client:** [Postman](https://www.postman.com/)
* **Scripting Language:** JavaScript (Postman Tests)
* **Testing Library:** Chai Assertion Library (Postman-рЃерЃў рЃўрЃюрЃбрЃћрЃњрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў)

## РџЎ№ИЈ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃњрЃљрЃЏрЃљрЃарЃЌрЃЋрЃљ (Setup)

1. **рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ:**
   рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ `test.json` рЃцрЃљрЃўрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃарЃЮрЃџрЃА. рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃЊрЃљрЃАрЃљрЃЦрЃЮрЃЦрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ:
   ```bash
   npx json-server --watch test.json --port 3000

   рЃАрЃћрЃарЃЋрЃћрЃарЃў рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃќрЃћ:http://localhost:3000

Postman-рЃўрЃА рЃЏрЃЮрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃљ: рЃџрЃЮрЃЎрЃљрЃџрЃБрЃарЃў рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃАрЃљрЃЉрЃљрЃќрЃўрЃАрЃЮ **URL (Base URL)** рЃДрЃЋрЃћрЃџрЃљ рЃарЃћрЃЦрЃЋрЃћрЃАрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА.

­ЪДф рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃАрЃфрЃћрЃюрЃљрЃарЃћрЃЉрЃў
рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃцрЃљрЃарЃњрЃџрЃћрЃЉрЃерЃў рЃарЃћрЃљрЃџрЃўрЃќрЃћрЃЉрЃБрЃџрЃўрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃбрЃћрЃАрЃбрЃћрЃЉрЃў:

1. рЃЉрЃљрЃќрЃўрЃАрЃБрЃарЃў рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ (Global Checks)
**Status Code Check: рЃЏрЃЮрЃгрЃЏрЃЊрЃћрЃЉрЃљ, рЃарЃЮрЃЏ рЃАрЃћрЃарЃЋрЃћрЃарЃў рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА 200 OK рЃАрЃбрЃљрЃбрЃБрЃАрЃА.**

**Performance Check: рЃърЃљрЃАрЃБрЃ«рЃўрЃА рЃЊрЃарЃЮ (Response Time) рЃљрЃа рЃБрЃюрЃЊрЃљ рЃљрЃдрЃћрЃЏрЃљрЃбрЃћрЃЉрЃЮрЃЊрЃћрЃА 200ms-рЃА.**

2. рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃА рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ
**Georgian Phone Format: рЃарЃћрЃњрЃБрЃџрЃљрЃарЃБрЃџрЃў рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃўрЃЌ (Regex) рЃЏрЃЮрЃгрЃЏрЃЊрЃћрЃЉрЃљ, рЃарЃЮрЃЏ рЃбрЃћрЃџрЃћрЃцрЃЮрЃюрЃўрЃА рЃюрЃЮрЃЏрЃарЃћрЃЉрЃў рЃерЃћрЃћрЃАрЃљрЃЉрЃљрЃЏрЃћрЃЉрЃљ рЃЦрЃљрЃарЃЌрЃБрЃџ рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃА (+995...).**

3. рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃўрЃА рЃбрЃћрЃАрЃбрЃћрЃЉрЃў (POST/PUT)
**Pre-request** рЃАрЃЎрЃарЃўрЃърЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ рЃ«рЃЊрЃћрЃЉрЃљ рЃЊрЃўрЃюрЃљрЃЏрЃўрЃБрЃарЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ, рЃарЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњрЃљрЃф рЃбрЃћрЃАрЃбрЃћрЃЉрЃў рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА:

**FirstName & LastName:** рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃерЃћрЃўрЃфрЃЋрЃљрЃџрЃљ рЃЌрЃБ рЃљрЃарЃљ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў рЃАрЃћрЃарЃЋрЃћрЃарЃќрЃћ.

**Full Name Validation:** рЃЏрЃЮрЃгрЃЏрЃЊрЃћрЃЉрЃљ, рЃарЃЮрЃЏ **name** рЃЋрЃћрЃџрЃў рЃАрЃгрЃЮрЃарЃљрЃЊ рЃљрЃарЃўрЃА рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў **FirstName**-рЃўрЃАрЃљ рЃЊрЃљ **LastName**-рЃА рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃљрЃфрЃўрЃўрЃЌ.

**Email & Phone Sync:** рЃЏрЃЮрЃгрЃЏрЃЊрЃћрЃЉрЃљ, рЃарЃЮрЃЏ рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃБрЃџрЃў рЃўрЃЏрЃћрЃўрЃџрЃў рЃЊрЃљ рЃбрЃћрЃџрЃћрЃцрЃЮрЃюрЃўрЃА рЃюрЃЮрЃЏрЃћрЃарЃў рЃќрЃБрЃАрЃбрЃљрЃЊ рЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљ рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃўрЃџ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА.
